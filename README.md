# مدرّب القواعد والنطق للآيلتس — IELTS Grammar & Writing Trainer

تطبيق محلي بالكامل (**offline-first**) لمستخدم واحد (أحمد)، لتمرين يومي قصير (١٥–٢٠ دقيقة) على أنماط أخطاء ثابتة بين جلسات المدرّب البشري. الهدف: باند **7.0** أكاديمي.

> **لا خدمات مدفوعة، لا مفاتيح API، لا خادم، لا حساب، لا سحابة.** كل شيء يعمل في المتصفّح والإنترنت مطفأ. جميع التمارين حتمية قائمة على قواعد (بلا ذكاء اصطناعي). ميزات «الذكاء الاصطناعي» تعمل بأسلوب **توليد برومبت جاهز تنسخه وتلصقه في Claude** — بلا أي تكلفة.

## كيف تشغّله

**على الكمبيوتر (الأسهل):** اضغط ضغطًا مزدوجًا على `index.html` — يفتح في المتصفّح ويعمل فورًا دون إنترنت ودون خادم.

> بيانات التمارين مكتوبة كملفّات JSON ملفوفة بسطر JS واحد (`window.BANK_... = [...]`) تحديدًا لتعمل بالنقر المزدوج بلا خادم.

### على الأندرويد والآيباد

**الطريقة ١ — عبر الواي-فاي المنزلي (بدون أي نشر):**
1. على الكمبيوتر: شغّل `start-server.bat` (نقرة مزدوجة). سيطبع لك عنوانًا مثل `http://192.168.1.5:8123`.
2. على الموبايل/الآيباد (نفس الشبكة): افتح هذا العنوان في Chrome أو Safari.
3. اترك نافذة الخادم مفتوحة أثناء التمرين.
> بهذه الطريقة يعمل التطبيق كاملًا، لكن **التثبيت على الشاشة الرئيسية والعمل أوفلاين** يتطلبان HTTPS — أي الطريقة ٢.

**الطريقة ٢ — GitHub Pages (مرة واحدة، ثم يعمل أوفلاين للأبد):**
انشر المجلد على GitHub Pages (مستودعك جاهز). بعدها من الموبايل/الآيباد افتح الرابط ثم:
- **أندرويد (Chrome):** قائمة ⋮ ← «إضافة إلى الشاشة الرئيسية» ← يتثبّت كتطبيق ويعمل أوفلاين.
- **آيباد (Safari):** زر المشاركة ⬆ ← «Add to Home Screen» ← يتثبّت ويعمل أوفلاين.

بعد التثبيت لا يحتاج الجهاز الإنترنت إطلاقًا — الـ service worker يخزّن كل شيء.

> **ملاحظة:** التقدّم (`localStorage`) لكل جهاز على حدة. لنقله يدويًا: «البيانات» ← **تصدير/استيراد** — أو الأفضل: فعّل **المزامنة التلقائية** أدناه.

## المزامنة التلقائية بين الأجهزة (مجانية)

تعمل عبر **GitHub Gist سرّي** على حسابك — صفر تكلفة، لا خادم، لا اشتراك:

- بعد كل إجابة يُرفع تقدّمك تلقائيًا (بعد ٤ ثوانٍ من آخر تغيير).
- عند فتح التطبيق على أي جهاز يُسحب التقدّم ويُدمج.
- **الدمج ذكي**: اتحاد سجلّي الجهازين وإعادة حساب الإحصائيات — لو تدرّبت على الموبايل والكمبيوتر بالتناوب، لا يضيع شيء من أي منهما.
- دون إنترنت؟ يشتغل عاديًا ويُزامن تلقائيًا عند عودة الاتصال.

**التفعيل (مرة واحدة على كل جهاز):**
1. أنشئ توكن من [github.com/settings/tokens](https://github.com/settings/tokens) ← *Generate new token (classic)* ← علّم صلاحية **gist فقط**.
2. في التطبيق: «البيانات» ← «المزامنة التلقائية» ← الصق التوكن ← **تفعيل**.
3. كرّر الخطوة ٢ بنفس التوكن على بقية أجهزتك — سيجد نفس الـ Gist ويكمل عليه.

> التوكن يُخزَّن محليًا على جهازك فقط. الـ Gist «سرّي» (لا يظهر في بحث GitHub)، ويحوي فقط إجاباتك وأسبابك — لا بيانات حساسة.

التقدّم وسجل الأسباب يُحفظان في `localStorage` بالمتصفّح على جهازك. استخدم زر **«تصدير»** في تبويب «البيانات» لأخذ نسخة احتياطية `.json` أو لمشاركة سجلّ تفكيرك مع مدرّبك.

## الفكرة الأساسية — «فكّر قبل أن تكشف»

لكل تمرين:
1. تظهر جملة بها فراغ (أو خيار متشابه).
2. تختار إجابتك.
3. **يجب** أن تكتب سببًا من سطر واحد («لماذا هذه الإجابة؟») قبل ظهور الحل.
4. عندها فقط تُكشف الإجابة الصحيحة + شرح مختصر + علامة صح/خطأ.
5. تُخزَّن **إجابتك وسببك** معًا لمراجعتهما لاحقًا (أنت ومدرّبك) في تبويب «البيانات».

لا يوجد اختصار لتخطّي كتابة السبب — بالتصميم.

## القواعد المُتتبَّعة (كلٌّ على حدة، بلا دمج للدرجات)

**القواعد (أنماط الخطأ الأربعة الأساسية):**
1. **الأدوات** — a / an / the / بدون أداة. مع تركيز على: «I have a car»، «advice» غير معدودة (لا «an advice»)، وقاعدة `of + اسم ⟵ the`.
2. **صيغة الفعل** — بعد حروف الجر (gerund) وبعد ضمائر الوصل (التطابق).
3. **الرابط والتطابق** — حذف is/are، و«the company **has**» لا «have».
4. **مفردات معكوسة المعنى** — قائمة قابلة للتوسعة (تبدأ بـ employer/employee).

**الصوتيات (طبقة المعرفة فقط — بلا صوت، بلا ميكروفون، بلا تقييم نطق):**
5. **نبر الكلمة** · 6. **نبر الجملة** · 7. **رموز IPA** · 8. **الشوا /ə/**
9. **المتلازمات اللفظية** (Lexical Resource) — مستخرجة ومنسّقة من *Cambridge English Collocations in Use*.

> **طبقة الصوت (النطق الفعلي، الميكروفون، التقييم) خارج نطاق هذا التطبيق تمامًا — تبقى في ELSA Speak.** هذا التطبيق يبني «الخريطة الصوتية» المعرفية فقط.

## الميزات

- **جلسة ١٥ دقيقة** — مجموعة مختلطة موزونة نحو أضعف قواعدك (الأقل دقّة يحصل على عناصر أكثر)، وتنتهي بشاشة ملخّص.
- **تدرّب على قاعدة واحدة** — عناصر من قاعدة بعينها.
- **لوحة التقدّم** — لكل قاعدة على حدة: دقّة % · محاولات · ستريك · آخر تدريب · مخطّط بصري. بلا «درجة إجمالية» مدموجة.
- **قاموس النطق الشخصي** — بطاقات (word · IPA · تقريب عربي · نمط النبر · سياق مالي)، قابلة للإضافة يدويًا.
- **مرجع المتلازمات** — قائمة بحث + إضافة أزواج كلمات متشابهة (تدخل الدرّيلات فورًا).
- **رسوم توضيحية** — مخطّطات SVG أوفلاين لكل قاعدة.
- **فحص الكتابة** — الصق مقالك ⟵ يولّد **برومبت تقييم كامل** (٤ معايير + وسم كل خطأ بأحد الأنماط الأربعة + كشف تضخّم Lexical Resource بسبب مفردات المال) تنسخه وتلصقه في Claude.
- **الترجمة** — حدّد أي نص إنجليزي داخل التطبيق واضغط **«ترجم المحدد»** (زر عائم)، أو الصق النص ⟵ يولّد برومبت ترجمة تلصقه في Claude. ومن الشاشة نفسها زر **«محتوى مرئي»** يولّد برومبت لإنشاء صورة/مخطّط تعليمي.

## أين تعيش البيانات وكيف تضيف

| ماذا | الملف | كيف تضيف |
|---|---|---|
| تمارين القواعد (أدوات/فعل/رابط) | `data/grammar.js` | أضف عنصرًا للمصفوفة بنفس الشكل |
| أزواج المعنى المعكوس | `data/vocab.js` | أضف زوجًا، أو من داخل تبويب «المرجع» |
| صوتيات (نبر/IPA/شوا) | `data/phonetics.js` | حسب القسم |
| قاموس النطق | `data/pron_dict.js` | أو من داخل تبويب «القاموس» |
| المتلازمات + المرجع | `data/collocations.js` | أضف عنصر درّيل و/أو مدخل مرجع |

كل ملفّ في `data/` مشروح بأعلاه بالشكل المطلوب لكل عنصر. مثال عنصر قاعدة:
```js
{ id:"art-99", rule:"articles",
  sentence_with_gap:"I need ___ pen.", correct_answer:"a",
  distractors:["an","the","(no article)"],
  explanation:"مفرد معدود غير محدّد ⇒ a." }
```
الإنجليزية تبقى LTR، والشرح عربي (RTL).

## البنية

```
index.html            ← يحمّل كل شيء بالترتيب
css/styles.css        ← RTL عربي + جزر LTR للإنجليزية، موبايل أولًا
data/                 ← بنوك العناصر (JSON ملفوف بسطر JS، قابلة للتحرير)
js/
  config.js           ← سجلّ القواعد + تجميع كل العناصر
  storage.js          ← localStorage: إحصاءات لكل قاعدة + سجل الإجابات/الأسباب + تصدير
  drills.js           ← محرّك «فكّر قبل الكشف» + التصحيح لكل الأنواع
  session.js          ← بناء الجلسة الموزونة + مشغّل الطابور + الملخّص
  diagrams.js         ← رسوم SVG لكل قاعدة
  dashboard.js        ← بطاقات الإحصاء + مخطّطات SVG
  prompts.js          ← مولّدات برومبت Claude (كتابة/ترجمة/صورة)
  app.js              ← الراوتر + التنقّل + كل الشاشات
```
